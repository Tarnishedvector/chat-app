# React & Firebase Chat App

![ChatApp UI](https://chat-app-ot6v.vercel.app/)

A modern, real-time chat application built with **React**, **Vite**, and **Firebase**. The application features a beautiful Glassmorphism design and animated UI, complete with public and private chat rooms.

### Live Demo
👉 **[View the Live Deployment](https://chat-app-ot6v.vercel.app/)**

### Features
- Real-time messaging using Firebase Firestore
- Secure Google Authentication
- Public Chat Room
- Password-Protected Private Room
- Auto-expiring messages (Messages delete after 24 hrs automatically using TTL)
- Fully responsive Glassmorphism User Interface
